Sistem Gereksinimleri

OpenCart SaaS altyapısının sorunsuz çalışması için sunucunuzun karşılaması gereken minimum ve önerilen teknik detaylar.

Sunucu Altyapısı #

SaaS platformumuz, en son PHP teknolojileri ve modern veritabanı motorları ile optimize edilmiştir. Performans kaybı yaşamamak için aşağıdaki sürümleri kullanmanızı öneririz.

lightbulb
İpucu

Daha yüksek trafikli mağazalar için Redis cache kullanımını şiddetle tavsiye ediyoruz.

PHP Versiyonları

Modüllerimizin çoğu PHP 8.1 ve üzeri özellikleri (Type hinting, Enums) kullanmaktadır.

  • Minimum: PHP 7.4
  • Önerilen: PHP 8.2
  • Gerekli Eklentiler: curl, gd, mbstring, zip, xml

Kurulum Komutları

Bağımlılıkları yüklemek için aşağıdaki Composer komutunu terminalinizde çalıştırabilirsiniz.

bash

composer require opencart-saas/core-module

php artisan migrate --seed

warning
Dikkat

Migration işlemi mevcut veritabanı tablolarınızda değişiklik yapabilir. Lütfen yedek almayı unutmayın.

Veritabanı Yapılandırması

MySQL 8.0 veya PostgreSQL 14+ sürümleri tam uyumluluk sağlamaktadır. Karakter seti olarak utf8mb4 kullanılması zorunludur.
